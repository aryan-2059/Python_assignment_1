# Daily Calorie Tracker

A simple Python-based Command Line Interface (CLI) tool to track daily calorie intake and save historical data.

## Features
- **Dynamic Input:** Allows users to specify the number of meals and details for each.
- **Calculation:** Automatically computes total daily calories and average calories per meal.
- **Health Monitoring:** Compares total intake against a user-defined daily calorie limit.
- **Data Persistence:** Option to save daily summaries to a `data.txt` file with a timestamp.

## Usage

1. Run the script:
   ```bash
   python lab_1.py

2. Enter the number of meals, meal names, and calorie counts when prompted.

3. Set your daily calorie limit.

4. Choose whether to save the data to the log file.
